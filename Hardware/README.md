# FCU
- UART1: Comm with NU
- UART2: Logging Debug
- UART3: Extra UART (Radio Control)
- UART4: Extra UART (Telemetry)
- UART5: Extra UART (Camera)

- I2C1: Comm with NU
- I2C2: Peripherals (PWM Driver)
- I2C3: EEPROM, Extra I2C

- SPI1: Peripherals (Flash)

- SWD: Coding/Debug

- SDIO: SD Card (1bit)

- GPIOA, GPIOC, GPIOE: Comm with NU (Output)
- GPIOB, GPIOD, GPIOF: Comm with NU (Input)

# NU
- UART1: Comm with FCU
- UART2: Logging Debug
- UART3: Peripheral (GNSS)
- UART4: Extra UART

- I2C1: Comm with FCU
- I2C2: Peripherals (Temperature, Baro, EEPROM, MAG)
- I2C3: Extra I2C

- SPI1: Peripherals (Flash)
- SPI2: Peripherals (IMU)

- SWD: Coding/Debug

- GPIOA, GPIOC, GPIOE: Comm with NU (Input)
- GPIOB, GPIOD, GPIOF: Comm with NU (Output)
